# Clarify Engine

> Локальный RAG-движок: задавай вопросы своим документам — получай точные ответы со ссылками на источники.

**Лицензия**: Apache 2.0  
**Уровень готовности**: Starter → Cloud Pro  
**Категория**: RAG / Поиск по документам  
**Статус**: Флагманское решение экосистемы open-ai.ru

---

## Задача

Малый и средний бизнес накапливает огромный объём корпоративных документов: регламенты, договоры, КП, письма, FAQ. Найти нужный ответ вручную — долго, а поиск по ключевым словам не понимает смысл вопроса.

Clarify Engine превращает ваши документы в базу знаний с разговорным интерфейсом: сотрудник задаёт вопрос на естественном языке и получает ответ со ссылкой на конкретный документ и абзац.

## Результат

- Время поиска ответа в корпоративных документах: с 10–30 минут до 10–30 секунд.
- Сотрудники перестают дублировать запросы в чатах — ответ находится самостоятельно.
- Ответы всегда обоснованы ссылкой на источник — нет «галлюцинаций».

## Целевые сценарии

- **Корпоративный FAQ**: новый сотрудник задаёт вопросы по регламентам и получает точные ответы с привязкой к разделу документа.
- **Юридическая база**: юрист или менеджер находит нужный пункт договора или нормативного акта за секунды.
- **Техподдержка**: агент поддержки мгновенно получает ответ из базы знаний продукта.
- **Анализ тендерной документации**: быстрое извлечение ключевых условий из объёмных тендерных файлов.

## Требования к среде

| Параметр | Starter (локально) | Business (сервер) | Cloud Pro |
|----------|--------------------|-------------------|-----------|
| RAM | 8 ГБ | 16 ГБ | 32 ГБ |
| CPU | 4 ядра | 8 ядер | 8+ ядер |
| GPU | не требуется | опционально | рекомендуется |
| Диск | 10 ГБ | 50 ГБ | 100+ ГБ |
| ОС | Linux / macOS / Windows (WSL2) | Linux | Linux |

На уровне **Starter** работает на обычном рабочем ноутбуке с 8 ГБ RAM без GPU, используя локальную LLM через Ollama (llama3.2, mistral-nemo и аналоги).

## Быстрый старт

```bash
# 1. Установить Ollama и скачать модель
curl -fsSL https://ollama.com/install.sh | sh
ollama pull llama3.2

# 2. Запустить Clarify Engine через Docker Compose
git clone https://github.com/open-ai-ru/clarify-engine.git
cd clarify-engine
cp deployment/.env.example deployment/.env
docker compose -f deployment/docker-compose.yml up -d
```

Открыть в браузере: `http://localhost:3000`

Загрузить документы (PDF, DOCX, TXT, MD) через интерфейс — через 1–2 минуты база знаний готова к вопросам.

## Ограничения

> Подробнее — в файле [limitations.md](limitations.md).

- На уровне Starter скорость ответа зависит от производительности CPU: 10–60 секунд на ответ без GPU.
- Максимальный размер одного документа на Starter-конфигурации: ~500 страниц.
- Обработка изображений и сканов (OCR) требует уровня Business+.
