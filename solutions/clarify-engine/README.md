# Clarify Engine — Флагманский RAG-движок экосистемы open-ai.ru

**Clarify Engine** — мощный открытый RAG-движок (Retrieval-Augmented Generation) с акцентом на высокую цитируемость ответов, безопасность данных и возможность локального развёртывания без зависимости от облачных сервисов.

## Ключевые преимущества

| Преимущество | Описание |
|---|---|
| **Высокая цитируемость** | Каждый ответ содержит ссылки на источники из вашей базы знаний |
| **Локальное развёртывание** | Полная работа без интернета — данные остаются у вас |
| **Безопасность данных** | Никаких утечек в облачные сервисы третьих сторон |
| **Гибкие модели** | Поддержка OpenAI, YandexGPT, Ollama и других LLM-провайдеров |
| **Open-source** | Полный исходный код, MIT-лицензия |
| **Масштабируемость** | От ноутбука до корпоративного кластера |

## Для каких задач подходит

- **Корпоративный поиск** по внутренней документации, регламентам, приказам
- **Клиентская поддержка** на основе базы знаний продукта
- **Юридическая аналитика** с точными ссылками на нормативные акты
- **Медицинские справочники** с цитированием источников
- **HR-ассистент** по политикам компании
- **Технический helpdesk** по документации продукта

## Уровни развёртывания

Clarify Engine доступен в четырёх конфигурациях под разные потребности и бюджеты:

| Уровень | Аудитория | Инфраструктура | Стоимость |
|---|---|---|---|
| [Starter](#starter) | Индивидуальный пользователь | Один компьютер | Бесплатно |
| [SMB](#smb) | Малый бизнес (до 50 человек) | Локальный сервер | Бесплатно |
| [Corporate](#corporate) | Средний бизнес (50–500 чел.) | Собственный сервер / VPS | Бесплатно |
| [Cloud Enterprise](#cloud-enterprise) | Крупный бизнес | Облачная инфраструктура | Бесплатно + расходы на облако |

Подробные инструкции — в файле [DEPLOYMENT.md](DEPLOYMENT.md).

## Быстрый старт

```bash
# Минимальный запуск (Starter-уровень)
git clone https://github.com/clarify-engine/clarify-engine
cd clarify-engine
cp .env.example .env
docker compose up
```

Через 2–3 минуты интерфейс доступен по адресу `http://localhost:3000`.

## Интеграция с экосистемой open-ai.ru

Clarify Engine спроектирован как центральный компонент экосистемы open-ai.ru и может использоваться совместно с другими решениями платформы через REST API.

Примеры оркестрации:
- **Clarify Engine + Open WebUI** — привычный чат-интерфейс поверх ваших данных
- **Clarify Engine + Agentic Tools** — автономные агенты с доступом к базе знаний
- **Clarify Engine + CRM** — умный ассистент в контексте клиентской базы

Подробнее — в файле [INTEGRATION.md](INTEGRATION.md).

## Требования

### Минимальные (Starter)
- CPU: 4 ядра
- RAM: 8 ГБ
- Диск: 20 ГБ
- Docker 24+, Docker Compose 2+

### Рекомендуемые (SMB/Corporate)
- CPU: 8+ ядер
- RAM: 16–32 ГБ
- Диск: 100+ ГБ SSD
- GPU (опционально): ускоряет локальные LLM-модели
