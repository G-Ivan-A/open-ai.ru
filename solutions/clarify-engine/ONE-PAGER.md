# Clarify Engine — One-Pager

## Задача

Сотрудники тратят часы на поиск ответов в корпоративных документах. Клиенты ждут ответов поддержки. Новички адаптируются неделями.

## Решение

**Clarify Engine** — open-source RAG-ассистент, который за 3 секунды находит ответ в ваших документах и называет точный источник. Работает локально — данные не покидают ваш сервер.

## Ключевые факты

- Ответ с цитатой из источника за **3–5 секунд**
- Поддержка **PDF, DOCX, Excel, HTML, Markdown**
- Локальное развёртывание — **данные только у вас**
- **Бесплатно** (MIT-лицензия, open-source)
- Работает **без интернета** (локальные LLM через Ollama)
- **Docker Compose** — запуск за 30 минут
