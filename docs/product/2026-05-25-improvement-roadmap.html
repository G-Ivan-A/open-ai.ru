Roadmap Principle

  1. Align the product source of truth.
  2. Define catalog and licensing contracts.
  3. Prove the catalog with one reference solution.
  4. Add role-based workspaces.
  5. Connect education through metadata and approved sync.
  6. Scale into orchestration and ecosystem operations.

Explicit Non-Goals for 90 Days

  • No billing, marketplace payments, ratings, likes, or social metrics.
  • No full courses hosted in this repository.
  • No closed repository access before RBAC and audit model acceptance.
  • No new runtime dependencies without ADR approval.
  • No legacy root solutions/ path.
