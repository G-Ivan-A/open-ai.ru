# AI Handover Prompt: open-ai.ru

Промпт передачи контекста между сессиями AI-агента. Его можно копировать в
новый диалог, чтобы агент быстро восстановил рабочий контекст репозитория.

## Готовый промпт

```text
Ты — AI-агент, работающий в репозитории open-ai.ru.

Перед изменением файлов выполни runtime-онбординг:

1. Прочитай README.md, AI_GOVERNANCE.md и AI_QUICK_RULES.md.
2. Прочитай текущую issue, последние комментарии и PR-контекст, если PR уже
   открыт.
3. Прочитай релевантные файлы в docs/, team/ и затронутом коде.
4. Кратко зафиксируй:
   - цель задачи;
   - Operating Mode;
   - границы и запреты;
   - план первых действий;
   - какие проверки подтвердят результат.
5. Если задача ясна и есть явное поручение продолжать, выполняй её до
   проверяемого результата.
6. Если контекста недостаточно, требования конфликтуют или изменение требует
   решения Founder, остановись и задай конкретный вопрос.

Работай минимально: не создавай документы, каталоги, зависимости и процессы,
которые не нужны текущей задаче.
```

## Когда использовать

- старт новой AI-сессии;
- передача задачи между агентами;
- возврат к PR после паузы;
- восстановление контекста после длинного обсуждения.
